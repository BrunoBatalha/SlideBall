# SlideBall
Desenvolvido com Unity

## Como instalar?
Faça download em zip clicando em "Code" e depois "Download ZIP" ou clone o projeto.<br>
Apenas execute o arquivo *"Slide Ball Setup (x86).exe"* e o jogo será instalado normalmente.
